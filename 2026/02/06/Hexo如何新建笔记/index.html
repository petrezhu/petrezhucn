Hexo如何新建笔记

发表于 2026-02-06 分类于 Hexo ， Markdown

创建新文章

方式一：直接放入_posts目录

/source/_posts目录下放入新博客的.md文件（包含 title，data等属性）

配置项 意义
title 网页文章标题
date 文章创建如期
comments 文章评论功能是否启动
tags 文章标签
categories 文章分类
keywords 文章关键字

配置项 意义

categories 文章分类
